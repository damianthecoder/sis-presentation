SIS Competitive Gaming · NBA production trial

A six-week, no-cost operating trial for NBA production reliability.

This suite proposes a bounded trial of format, scheduling and visible-progression changes on NBA Night production, run at existing pay and existing staffing, to test reliability as a revenue lever before any investment decision is made.

The trial tests the reliability lever before any spend is proposed.

Cancellations behave like a controllable rate, not a fixed cost of the current format. A six-week run on Studios 5-6 asks whether format, stagger and visible progression can lower that rate at existing pay and existing staffing. A phase-2 bonus and multiplier model — with an estimated 0.78-percentage-point break-even — is proposed only after the trial's gates pass.
